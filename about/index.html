Features:

  • Fully installable, right from your browser
  • Dark mode and light mode (Dark mode is default.)
  • Auto Sudoku generation
  • Works offline (After opening the installed version at least once with internet.)
  • Multiple difficulties (Easy, Medium, Hard and Expert.)
  • Hints (Currently restores the board rather than using solving techniques.)
  • Timer
  • Pencil mode
  • Erase mode
  • Game saving
  • Theme preference saving (Pick between dark and light mode, then have the game remember which mode you picked and switch to it next time you open the app.)
  • No ads
  • Pause on lost focus (Your game will be paused if you minimize, close, or otherwise make the app not visible.)

Where to find it:

Ceedoku is a website, not an app. However, this does not mean you cannot install Ceedoku. Below are all the ways you can access Ceedoku.
